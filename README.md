# plugins-installer
自动安装koishi.yml中的插件

[![npm](https://img.shields.io/npm/v/koishi-plugins-installer?style=flat-square)](https://www.npmjs.com/package/koishi-plugins-installer)


## 环境要求

### 1.node
### 2.koishi

## 使用方法

### 1.将installer.js拷贝到koishi的根目录下

### 2.命令行切换到koishi根目录

### 3.运行以下命令
`node installer.js`
